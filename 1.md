# Prompt 1 — Project Kickoff

## Request

Build a web tool module for polar sound distribution measurement:

- **5 UMIK-2 microphones** arranged in a vertical arc at -90°, -45°, 0°, +45°, +90° elevation
- Capture raw audio from all 5 mics simultaneously at each azimuth position
- User manually rotates object on turntable, enters azimuth angle, clicks capture
- Plot polar distribution of sound (SPL) across the measurement sphere
- Export captured audio as WAV files

## Decisions Made

| Question | Answer |
|----------|--------|
| Integration | Standalone but portable (embeddable later) |
| Mic layout | Vertical arc at fixed elevations |
| Data type | Raw audio capture |
| Workflow | Single snapshot + guided sequence (future) |
| Elevations | -90°, -45°, 0°, +45°, +90° |
| Turntable control | Manual rotation, user enters angle |
| Tech stack | React + Vite + TypeScript + Tailwind |
| Audio format | WAV files (browser download) |
